# react-sliderInput
A slider(range) input React.js module

## Usage
to build, simply do this in node:
```
npm install
webpack
```
then just open `build/index.html` to see the demo.

## Options
#### `size`
- Number
- Default: 400

the size of the slider in pixel

#### `initialProgress`
- Number
- Default: 0

the initial progress

#### `min`
- Number
- Default: 0

#### `max`
- Number
- Default: 0

#### `indicate`
#### `step`
#### `editable`
#### `onChange`
	size: 400,
	min: 0,
	max: 100,
	indicate: false,
	step: 1,
	editable: true,
	onChange: nullfn,
